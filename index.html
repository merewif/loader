Preloader using CSS pseudo-elements and JavaScript. The element is a div with rounded corners, the two other spinning lines are its ::before and ::after elements. Adjacent borders give the spinning lines a sharp edge and different animation durations for all 3 lines add a positively chaotic feel to the design.

function hide() { document.getElementById('loaderbox').style.display = 'none'; document.getElementById('loader').style.display = 'none'; document.getElementById('logo').style.visibility = 'visible'; document.getElementById('content').style.visibility = 'visible'; document.getElementById('background').style.visibility = 'visible'; } window.addEventListener ("load", function() { setTimeout(hide, 3000); })

Hover effect utilizing CSS pseudoelements and scaling transformation on the Y axis for horizontal effect or on the X axis for vertical effect. A "useless" translateZ() function was added so the element gets its own composite layer.

.upper { transform: perspective(0) translateZ(0); } .upper:before { content: ""; transition-property: transform; transform: scaleY(0); transition-duration: 0.25s; transition-timing-function: ease-in-out; } .upper:hover:before { transform: scaleY(1); }
